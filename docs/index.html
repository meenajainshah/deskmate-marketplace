v0.1
CONTRACTS

Contract Guard

Reviews recruiting contracts in plain English and flags the clauses that cost recruiters money.

  • Fall-off / guarantee & fee trigger
  • Candidate ownership & non-circumvent
  • Uncapped indemnity & missing protections
  • Redlines + marked-up DOCX
v0.2
GET PAID

Fee Chaser

Chases unpaid placement fees — but only the ones that are actually overdue, per each client's real terms.

  • Reads net terms & when the fee is earned
  • Leaves on-time clients alone
  • Tone-matched reminders you approve
  • Never sends or charges on its own
v0.3
FILL ROLES

Job Kit

Turns a client's rough req into a full working kit — priced against the same terms.

  • Polished, un-inflated job description
  • Structured interview guide + 1–5 rubric
  • Offer-letter template (draft only)
  • Priced up front from the Deal Record
Install

Add it in two lines

In Claude Code (or Cowork's terminal), add the marketplace and install the plugin:

/plugin marketplace add meenajainshah/deskmate-marketplace
/plugin install deskmate@deskmate-marketplace

Then run /deskmate:contract-guard — or just upload a contract and ask "any red flags?"

Design principle

Own the core, connect for the rest

Deskmate owns the recruiting-specific spine and connects to your existing tools for the generic back office — it will never be an accounting package or an ATS.

✅ Deskmate builds

  • The Deal Record (recruiting terms)
  • Recruiting-clause review
  • "Is this fee actually overdue?" logic
  • Guarantee / fall-off & backdoor-hire exposure
  • Role kits (JD · rubric · offer)

🔌 Deskmate connects to

  • Invoicing, payments, cash forecast → QuickBooks / Stripe
  • Email → Gmail / Outlook
  • Candidate pipeline → your ATS / CRM
  • Connectors are optional enrichment
  • Solo works fully standalone
